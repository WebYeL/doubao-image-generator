## [1.0.0] - 初始版本

### 功能
- 基于 Doubao-Seedream-5.0-lite 的 AI 图片生成应用
- 支持文生图和图生图功能
- 后端使用 FastAPI + Python
- 前端使用 Vue 3 + Ant Design Vue
- 支持多种图片尺寸（1K/2K/4K/竖图）
- 生成历史记录管理
- 图片预览和下载功能
